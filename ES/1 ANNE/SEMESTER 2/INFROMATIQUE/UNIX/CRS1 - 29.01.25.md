## Ex 2

| **Commande**    | **Description**                                                               |
| --------------- | ----------------------------------------------------------------------------- |
| `w`             | Affiche les utilisateurs connectés, leurs processus et la charge système.     |
| `who`           | Liste les utilisateurs actuellement connectés au système.                     |
| `whoami`        | Affiche le nom d'utilisateur de la session actuelle.                          |
| `users`         | Liste les noms des utilisateurs connectés.                                    |
| `tty`           | Indique le nom du terminal associé à la session actuelle (ex: `/dev/pts/10`). |
| `id`            | Affiche les identifiants de l'utilisateur (UID, GID et groupes).              |
| `logname`       | Montre le nom de l'utilisateur ayant ouvert la session.                       |
| `date`          | Affiche la date et l'heure actuelles.                                         |
| `hostname`      | Indique le nom de la machine (ex: `lozan`).                                   |
| `df`            | Liste l’espace disque utilisé et disponible sur les systèmes de fichiers.     |
| `clear`         | Efface l’écran du terminal.                                                   |
| `pwd`           | Affiche le répertoire de travail actuel (ex: `/home/zeinalovd`).              |
| `echo $LOGNAME` | Affiche la variable d’environnement `LOGNAME` (nom de l’utilisateur).         |
| `echo $TERM`    | Affiche la variable d’environnement `TERM` (type de terminal utilisé).        |
| `printenv`      | Liste toutes les variables d’environnement.                                   |
| `ps`            | Affiche les processus en cours dans la session actuelle.                      |
| `ps -a`         | Affiche tous les processus, y compris ceux des autres utilisateurs.           |
| `ps -ax`        | Liste tous les processus du système (même sans terminal).                     |
| `ps -alx`       | Affiche les processus avec des détails étendus (priorité, UID, etc.).         |
| `ls`            | Liste les fichiers et dossiers du répertoire courant.                         |
| `ls -a`         | Liste tous les fichiers, y compris les fichiers cachés (commençant par `.`).  |
| `ls -l`         | Affiche les fichiers avec des détails (permissions, taille, date).            |
| `ls -al`        | Combine `-a` et `-l` pour afficher tous les fichiers avec détails.            |
| `history`       | Affiche l’historique des commandes exécutées dans la session.                 |
